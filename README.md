# 📚 Library Management System Frontend

A modern frontend application for managing a book library, allowing users to add, edit, and delete books with real-time API interactions.

---

## 🚀 Features

- ➕ Add a new book and Borrow a book
- 📄 View all books & borrow in a card layout
- 🗑️ Delete individual books
- ⚡ Real-time API communication with RTK Query
- 🌐 Routing for pages like /books, /create-book, /borrow-summary etc.
- 🍞 Toast notifications for success/error actions
- 💻 Responsive design with Tailwind CSS
- 🧠 Built with TypeScript for type safety

---

## 🛠 Technologies

- ⚛️ React + Vite
- 🧰 Redux Toolkit + RTK Query
- 🔧 TypeScript
- 🚦 React Router DOM
- 🍞 React Toastify
- 💨 Tailwind CSS
